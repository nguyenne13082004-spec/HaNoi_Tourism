I. Danh sách các chủ đề dự án thiết kế website (Tin học 12)
Website giới thiệu du lịch địa phương
II. Yêu cầu dự án:

1. Sản phẩm website:
   • Có ít nhất 4-5 trang (Trang chủ + các trang nội dung)
   • Có menu điều hướng, link hoạt động đúng
   • Nội dung đúng chủ đề, rõ ràng
2. Yêu cầu kỹ thuật: Website phải có:
   • Hình ảnh, video
   • Liên kết
   • Danh sách (list)
   • Bảng hoặc form
   • Có sử dụng CSS
3. Trình bày
   • Bố cục rõ ràng
   • Màu sắc hài hòa, dễ đọc
   • Các trang thống nhất
4. Nộp bài: Sản phẩm nộp gồm:
   • 1 thư mục gồm:
   o File .html
   o File .css
   o Thư mục ảnh
   • 1 file mô tả ngắn (Word/PDF):
   o Tên website
   o Ý tưởng
   o Các nội dung chính
   o Phân công nhiệm vụ
5. Về thuyết trình
   • Thời gian: 3–5 phút
   • Nội dung:
   o Giới thiệu website
   o Demo các trang chính
   o Nêu điểm nổi bật
   III. CÁCH CHẤM ĐIỂM
   Tiêu chí Điểm
   Nội dung 2
   Cấu trúc (số trang, menu, link) 2
   Giao diện (CSS, trình bày) 2
   Thành phần HTML (ảnh, bảng/form…) 2
   Sáng tạo 1
   Thuyết trình 1
